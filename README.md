# OnlyForU Creator Marketing Website

Live marketing website for OnlyForU - The Resolution Economy platform for creators.

## Features
- Modern, responsive design
- AI-powered earnings calculator
- Creator signup form with Formspree integration
- Beautiful O+U logo with flowing connection design

## Deployment
Automatically deployed to Azure Static Web Apps on push to main branch.

## Local Development
Simply open `index.html` in a browser to view locally.

## Files
- `index.html` - Main website
- `styles.css` - All styling
- `script.js` - Calculator and form functionality
- `logo.png` - OnlyForU logo
